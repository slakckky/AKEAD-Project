# AKEAD Invoice Matcher

AKEAD şirketi için fatura (invoice) tarama ve barkod/referans eşleştirme projesi.

## Amaç (taslak)

- PDF faturaları taratıp (scan) içeriğini extract etmek.
- Faturadaki barkod numarası ile referans numarasını yapay zeka destekli eşleştirme yaparak ilişkilendirmek.
- Eşleştirme sonucunda hiçbir barkod numarasının boşta (eşleşmemiş) kalmamasını sağlamak.

## Durum

Proje henüz başlangıç aşamasında. Detaylı gereksinimler, teknoloji yığını ve mimari kararlar sonraki adımda netleştirilecek.

## Notlar

- Detaylar netleştikçe bu dosya güncellenecek.
